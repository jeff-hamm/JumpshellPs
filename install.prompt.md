# Install Jumpshell Extension

Download and run the full installation instructions from GitHub.

Raw URL: `https://raw.githubusercontent.com/jeff-hamm/jumpshell/main/install-details.prompt.md`
Temp file: `<TEMP_DIR>/jumpshell/install-details.prompt.md`

1. Create `<TEMP_DIR>/jumpshell` if it does not exist.
2. Download `https://raw.githubusercontent.com/jeff-hamm/jumpshell/main/install-details.prompt.md` to `<TEMP_DIR>/jumpshell/install-details.prompt.md`.
3. Confirm the downloaded file starts with `# Install Jumpshell Extension`.
4. Run `<TEMP_DIR>/jumpshell/install-details.prompt.md` as a prompt.
